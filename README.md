# To-Do List Web Application

A simple task management web application built using **PHP and MySQL**.
This application allows users to manage daily tasks by adding, editing, viewing, and deleting tasks.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* View all tasks in a list

## Technologies Used

* HTML
* CSS
* PHP
* MySQL
* WAMP Server

## How to Run the Project

1. Start **WAMP Server**.
2. Place the project folder inside the `www` directory.
3. Create the database `todo_app` in **phpMyAdmin**.
4. Import the required table structure.
5. Open your browser and go to:

```
http://localhost/todo-list-project
```
